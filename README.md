# Maschera di Venezia

Sito statico pronto per GitHub Pages.

## Struttura
- `index.html` → pagina principale con tutti i capitoli
- `capitoli/` → una pagina HTML per ogni capitolo
- `assets/css/styles.css` → stile del sito
- `assets/js/main.js` → ricerca nella homepage
- `assets/images/` → immagini del banner e dei capitoli

## Immagini
Per usare le immagini, metti i file in `assets/images/` con questi nomi:

- `banner.jpg`
- `capitolo-01.jpg`
- `capitolo-02.jpg`
- ...
- `capitolo-30.jpg`

## Pubblicazione su GitHub Pages
1. Carica tutti i file in un repository GitHub.
2. Vai su **Settings → Pages**.
3. In **Build and deployment**, scegli il branch principale e la cartella root.
4. Salva.

Dopo pochi istanti, il sito sarà online.
